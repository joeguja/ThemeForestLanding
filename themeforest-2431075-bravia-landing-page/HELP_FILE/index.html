A) HTML Structure - top

Bravia Landing Page is a theme which fixed layout with up to 12 columns. All of the information within the main content area is nested within a div with an id of "primaryContent".The general template structure is the same throughout the template as it uses the 12 column 960.gs framework.

The general concept behind the 12 column framework is that all the columns (labeled grid_x) need to add up to 12 when using the 12 column version of their framework. The best way I would suggest to become familiar with this framework is to simply edit what I have already set up. Here is the general structure.


If you would like to edit the color, font, or style of any elements on the site please find the element's class or ID and locate where it is being controlled in one of the CSS files.

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

B) CSS Files and Structure - top

There are some CSS Files used in this theme. And here is the short explanation about them :

  • 960_12_col.css - The 960gs Grid Framework, do not edit this file.
  • text.css - Text CSS for 960 Grid System, you don't need to edit this
  • style.css - This is the main CSS file that controls almost everything on the site EXCEPT some attributs on slider, prettyPhoto, and tooltip
  • reset.css - Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.
  • prettyPhoto.css - Pretty Photo CSS, you should not need to edit this file
  • diapo.css - Slider CSS, you should not need to edit this file
  • video-js.css - Self hosted Video CSS, you should not need to edit this file

C) JavaScript - top

All the pages link the neccessary js files inside the "head" tag of the document. If you need to modify the behavior from a certain jquery element open the "js/custom.js" file. where you can alter the parameters of some of the js elements.

  1. jQuery - a Javascript library that greatly reduces the amount of code that you must write. For more information, please visit http://www.jquery.com/
  2. Diapo Slider - a jQuery slideshow with many effects, transitions, easy to customize, using canvas and mobile ready, based on jQuery 1.4+. In this theme, Diapo Slider used on Featured item. Refer to the author's documentation for additional notes on how to use it.
  3. PoshyTip - a jQuery tooltip plugin. In this theme, Poshytip used on the input forms. Refer to the author's documentation for additional notes on how to use it.
  4. prettyPhoto - a jQuery plugin to enhance the image/photos/gallery. In this theme, prettyPhoto used on the Recent Work/portfolio. Refer to http://www.no-margin-for-errors.com for more info.
  5. Testimonial Cycle - a jQuery plugin to animate/cycle the testimonial. You can also visit http://www.jquery.com/ for more info about jQuery Cycle.
  6. Video JS - a jQuery plugin to make a self hosted video using HTML5 Semantics. You can also visit http://www.videojs.com/ for more info about Video JS.
  7. Retina JS - a jQuery plugin to make a retina ready template. You can also visit http://www.retinajs.com/ for more info about Retina JS.

D) PHP Files - top

This template has two php files, they are registration.php for Registration/Sign Up Form and mailchimp-subscriber.php for Mailchimp Subscribtion Form. All files must be updated before use. Here is how it works:

  1. If user hit the "Sign Me Up Now" button, then the form submitted to registration.php which will send you and the user a notification mail
  2. If user hit the "Get Notified" button, then the form submitted to mailchimp-subscriber.php. Of course you have to registered to mailchimp and have an API key. If you have some question you can also check the reference here
